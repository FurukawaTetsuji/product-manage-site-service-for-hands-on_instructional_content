# バージョンアップについて

ng update

## アップデートの有無を知りたい時

ng update だけ

## 実際に上げたい時

ng update @angular/core

ng update @angular/cli

ng update @angular/material

ng update @angular/cdk

## 強制で上げたい時

ng update @angular/core --force
